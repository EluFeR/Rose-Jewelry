# 💍 Rose Jewelry — E-Commerce Web App

A fully responsive jewelry e-commerce platform built with HTML, CSS, and JavaScript, featuring product listings, a shopping cart, checkout flow, user authentication, and a blog.

## 🛍️ Features

- **Product Catalog** — Browse and filter jewelry items with a dedicated detail page
- **Shopping Cart** — Add/remove items, view cart summary
- **Checkout Flow** — Multi-step checkout with form validation
- **User Authentication** — Login page with session handling
- **Blog** — Informational posts with individual detail pages
- **Responsive Design** — Mobile-first layout across all pages

## 🗂️ Pages

| Page | Description |
|------|-------------|
| `index.html` | Homepage / landing |
| `product.html` | Product listing |
| `detail.html` | Single product detail |
| `cart.html` | Shopping cart |
| `checkout.html` | Checkout form |
| `login.html` | User login |
| `about.html` | About page |
| `blog.html` | Blog listing |
| `blog_detail.html` | Single blog post |
| `contact.html` | Contact form |

## 🧰 Tech Stack

- HTML5, CSS3, JavaScript (ES6+)
- Custom fonts & responsive CSS grid/flexbox
- Git & GitHub for version control
